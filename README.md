# Mobile UI - Balatro Mod

A Steamodded mod that improves readability, touch usability, and visual hierarchy for mobile/Android Balatro while preserving the game's retro pixel-art identity.

## Features

- **Larger text** — Small labels, blind descriptions, and utility text scaled up 15%
- **Bigger touch targets** — All buttons increased in size with more padding
- **Better contrast** — Brighter foreground text, darker backgrounds
- **Thinner outlines** — Reduced border clutter without losing the pixel-art look
- **More spacing** — Blind choices, HUD sidebar, and interactive elements get breathing room
- **Color hierarchy** — Secondary elements muted slightly, key elements stay vibrant
- **Darkened background** — Green pattern background dimmed so cards pop more

## All Features Are Toggleable

Adjust via the config tab in Steamodded's mod settings.

## Requirements

- Balatro (any version with Steamodded support)
- Steamodded 1.0.0+
- Lovely injector (for Android builds)

## Installation

Copy `mobile-ui.lua` and `config.lua` to your Balatro `Mods/mobile-ui/` directory.
